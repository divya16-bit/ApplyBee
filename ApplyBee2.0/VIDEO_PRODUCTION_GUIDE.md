# ApplyBee Promo Video Production Guide

## Quick Start Options

### Option 1: DIY with Screen Recording (Free)
**Tools Needed:**
- OBS Studio (free screen recorder)
- Canva (free video editor)
- Your Chrome extension

**Steps:**
1. Record screen while using the extension
2. Edit in Canva with text overlays
3. Add background music (free from YouTube Audio Library)
4. Export as MP4

### Option 2: Hire a Professional (Recommended)
**Platforms:**
- **Fiverr:** Search "chrome extension promo video" ($50-200)
- **Upwork:** Post job for video editor ($100-500)
- **99designs:** Contest-based ($299+)

**What to Provide:**
- Script (included in PROMO_VIDEO_SCRIPT.md)
- Screenshots/video clips of extension
- Logo/icon files
- Brand colors (black #1a1a1a, yellow #ffd700)

### Option 3: AI Video Tools
**Tools:**
- **Synthesia:** AI avatars + voiceover ($30/month)
- **Pictory:** Auto-create from script ($19/month)
- **InVideo:** Template-based ($15/month)

## Video Specifications for Chrome Web Store

**Required:**
- **Promotional Video:** 1 video (optional but recommended)
- **Format:** MP4, WebM, or MOV
- **Max Size:** 120MB
- **Resolution:** 1280x720 or higher
- **Duration:** No specific limit, but 60-90s recommended

**Best Practices:**
- Show actual extension in use
- Highlight key features
- Include call-to-action
- Professional quality
